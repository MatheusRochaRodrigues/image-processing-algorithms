## Image Processing Algorithms

This repository contains implementations of several image processing algorithms developed for study and experimentation.

### Images

The images used for testing the algorithms are located in the `Images/` folder.

### Results

The outputs generated by the algorithms are stored in the `Results/` folder.
These images demonstrate the effect of each algorithm applied to the input images.

### Example Workflow

1. Select an image from the `Images/` folder
2. Run the desired algorithm
3. The processed image will be saved in the `Results/` folder

### Repository Structure

```
image-processing-algorithms/
│
├── Images/      # input images used for testing
├── Results/     # processed images generated by the algorithms
├── src/         # algorithm implementations
└── README.md
```
